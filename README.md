# CineDock
CineDock is a web wrapper around movie streaming site but it will help you ignore all ads and popups and risk of being traced or malware. ENJOY STREAMING

CineDock is a lightweight Android wrapper built for fast browsing, quick search, bookmarks, and a cleaner mobile viewing experience.

## Features

- Clean web-based viewing experience
- Floating assistive ball for quick actions
- In-app search
- Bookmark and saved items support
- Sign in with email/password
- Sign in with Google
- Guest access
- Profile display name editing
- Firebase-backed sync for user data and bookmarks
- FCM notifications
- Remote Config support for updates and maintenance messages
- Update checker with APK download prompt
- Custom app branding and splash screen

## Getting Started

1. Install the app.
2. Sign in with Google, email/password, or continue as guest.
3. Use the floating button for quick access to:
   - Search
   - Bookmarks
   - Settings
   - Current page bookmark toggle

## Search

Use the search option from the floating menu to search directly from inside the app or on the top left corner for accurate results.

## Account Options

CineDock supports:
- Google sign-in
- Email and password sign-in
- Guest mode

You can also update your display name from the profile screen.

## Bookmarks

Save pages you want to return to later. Bookmarks are synced for signed-in users.

## Notifications

CineDock supports Firebase Cloud Messaging for alerts, updates, and maintenance notices.

## Updates

The app can check for newer versions through Firebase Remote Config and prompt the user to download an update APK.

## Permissions

Depending on your Android version and usage, the app may request:
- Internet access
- Network state access
- Notifications
- Storage access for older Android versions when needed for downloads
